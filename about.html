Vrbic was founded in 2022 with a singular goal of creating a simple guide on building a small e-commerce site. Vrbic is based on jekyll, a static site generator. It's easy to use and easy to create with. the project was created using jekyll create site cli command. The site also used snipcart addon as a main e-commerce option. Hosting and deploying is provided by netlify and the code is kept on gitlab. Further text will be added...
